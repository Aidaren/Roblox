# **Advanced StepSound (All material stepsound)**

## 中文简介/Chinese Language
**恭喜你发现了这个宝藏脚本，此脚本的宗旨皆在为真实的游戏体验所服务  
此脚本通过判断不同的地形来改变你的脚步声，并且具有高度可自定义性！**
### **脚本功能：**
* **不同材质下的不同脚步声**
* **不同速度下的脚步声快慢**
* **在速度更改后保持当前脚步声所有参数**
* **高度自定义**
